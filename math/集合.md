# 集合

## 定义
### 元素(element)
元素:就是个简单的东西
- **∈属于(belongs to)** a∈A: a是集合A的元素
- **∉不属于(not belongs to)** a∉A: a不是集合A的元素
### 集合(set)
**集合**: 一堆元素(a set o' 元↑素↓)
奇奇怪怪的表示符号:
- **∅空集**(empty set): 没元素 
- **非空集**(non-empty set): 不是空的,有元素
- **⊆子集**(subset): 所有元素右边集合都有
- **⊂真子集**(proper subset): 所有元素右边集合都有, 但是右边集合**绝对**有更多元素
- **∪并集**(union): 两个集合所有的
- **∩交集**(intersection): 两个集合都有的 (画个U当集合,再画一下符号当第二个集合,就分得清交集和并集了,方向一样就是一并,方向不一样就是相交)
- **−差集**(difference): 一个集合中减第二个集合后剩下的
- **补集**(complement): 这个集合中没有的元素
- **|A|** 集合A的元素个数

**表示集合**:
A = {1,2,3}
B = {x∈ℝ|x<114514}

**常用集**:
- **ℕ自然数**(natural numbers): 0,1,2,3,4,5...
- **ℕ*(ℕ+)正整数**(natural numbers): 1,2,3,4,5...(没有0)
- **ℤ整数**(integers): ...,-3,-2,-1,0,1,2,3...(int有负数)
- **ℚ有理数**(rational numbers): 能写出来的
- **ℝ实数**(real numbers): 所有的数, 包括无理数(写不出来的,比如π,√2)
- **U全集**(universal set): 当前所有的元素
