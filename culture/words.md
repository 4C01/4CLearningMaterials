# 多语言字母对照表 (西里尔/阿拉伯/拉丁/汉语音标)

> **使用说明**：
> - 阿拉伯字母因连写规则，同一字母在词首、词中、词尾形态不同，表中均以 **单独/词首/词中/词尾** 顺序列出（若有变化）。
> - 汉语音标采用拼音近似+描述，部分特殊发音用国际音标[ ]补充。
> - 哈萨克语、维吾尔语、俄语、乌克兰语均以各自现行标准文字为准。

### 4. 维吾尔语特殊字母 (阿拉伯)

> **书写规则**：每行格式为 **单独/词首/词中/词尾**，若四态相同则只写一次。

| 阿拉伯 | 拉丁 | 西里尔 (维) | 汉语音标/发音方式 | 四态写法 |
|--------|------|-------------|-------------------|-----------|
| ئە | E e | Ə ə | 额 [ɛ] | **ئە / ئە / ـە / ﯥ** |
| ئى | I i | И и | 伊 [i] / 额 [ɨ] | **ئى / ئى / ـى / ﻰ** |
| ۇ | U u | У у | 乌 [u] | **ۇ / ۇ / ـۇ / ﻮ** |
| ۈ | Ü ü | Ү ү | 淤 [y] | **ۈ / ۈ / ـۈ / ﻮ** |
| ۆ | Ö ö | Ө ө | 欧圆唇 [ø] | **ۆ / ۆ / ـۆ / ﻮ** |
| ۋ | W w | В в | 瓦 [w] | **ۋ / ۋ / ـۋ / ﻮ** |
| ﯞ | V v | В в (某些方言) | 弗 [v] | **ﯞ / ﯞ / ـﯞ / ﻮ** |
| ق | Q q | Қ қ | 小舌柯 [q] | **ق / ق / ـق / ﻖ** |
| غ | Gh gh | Ғ ғ | 小舌赫 [ʁ] | **غ / غ / ـغ / ﻎ** |
| ھ | H h | Һ һ | 喉呼 [h] | **ھ / ھ / ـھ / ﮫ** |
| نٛ | Ng ng | Ң ң | 后鼻音 [ŋ] | **نٛ / نٛ / ـنٛـ / ﻨٛ** |

> **维吾尔语特别说明**：
> - **ئە** 和 **ئى** 当出现在词首时，必须加 **ئ** (hamza) 作为辅音支撑。
> - **نٛ** 是 ن + 上标小 nun，部分地区用 **ڭ** 代替。
> - 元音 **ئۇ / ئۈ / ئۆ** 词首也必须加 **ئ**。
