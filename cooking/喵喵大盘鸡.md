# 家常喵喵大盘鸡

以鸡腿肉为主料，搭配土豆、青椒，炒糖色后炖煮，香浓微辣，可配米饭或皮带面，是一道扎实的新疆风家常菜。

## 前期准备

| 材料 | 用量 | 作用与选择 |
| :--- | :--- | :--- |
| 鸡腿 | 3个（约500g） | 去骨后切小块，肉质嫩滑 |
| 土豆 | 1–2个（约300g） | 切厚片或滚刀块，炖至软糯 |
| 青椒 | 1个 | 切块，出锅前增色提味 |
| 葱 | 2根 | 一部分焯水用，一部分切段最后用 |
| 姜 | 1块（约20g） | 切片，分焯水和炒制用 |
| 干辣椒（红辣皮子） | 5颗（可根据辣度调整） | 增香微辣 |
| 花椒粒 | 一小撮（约20粒） | 增麻香 |
| 八角 | 1颗 | 增香 |
| 冰糖 | 15–20g | 炒糖色，使鸡肉红亮 |
| 辣椒酱 | 1勺（约15g） | 增加复合辣味 |
| 生抽 | 1勺（约10ml） | 提鲜调味 |
| 老抽 | 1勺（约10ml） | 上色 |
| 蚝油 | 半勺（约5ml） | 增鲜 |
| 料酒 | 2勺（约20ml） | 焯水去腥 |
| 盐 | 适量（约3–5g） | 最后调味 |
| 鸡精 | 2勺（约6g） | 提鲜 |
| 白糖 | 一小撮（约2g） | 平衡咸味 |
| 植物油 | 适量 | 炒糖色及煎土豆用 |

## 制作方式

1. **处理鸡肉**  
   鸡腿去骨，切成2–3厘米的小块。冷水下锅，加入**葱段、姜片、料酒**，大火煮沸后撇去浮沫，捞出鸡肉，用**热水**冲洗干净（冷水会使肉质收缩变硬），沥干备用。

2. **炒糖色**  
   锅中倒少许油，放入冰糖，小火加热至冰糖融化并呈琥珀色（微微冒泡），迅速倒入焯好的鸡肉，翻炒均匀，使鸡肉裹上糖色。

3. **煸香调味**  
   加入**姜片、花椒粒、八角**，继续翻炒至鸡肉表面微干、水分收干。放入**1勺辣椒酱**炒出红油，再淋入**生抽、老抽、蚝油**，翻炒均匀。

4. **炖煮**  
   加入足量热水，没过鸡肉即可。放入**干辣椒**，盖上锅盖，中火焖煮**30分钟**。

5. **准备土豆**  
   在炖煮最后5分钟前，将土豆切厚片或滚刀块。  
   *可选*：锅中放少许油，将土豆两面煎至微微焦黄，再入锅炖煮，口感更香且不易散。

6. **合炖收尾**  
   将土豆放入锅中，继续炖煮**5–10分钟**（根据喜欢的软烂程度调整）。  
   加入**盐、鸡精、少许白糖**，放入**葱段、青椒块**，大火翻炒1–2分钟，汤汁收至浓稠即可出锅。

**总制作时间约50分钟（不含准备）**

## 食用方式

大盘鸡适合搭配**馍馍**或**皮带面**食用。将皮带面煮熟过凉水，铺在盘底，浇上鸡肉和汤汁，面条吸足汤汁更美味。

### 皮带面做法（可选）
- 购买现成面剂子，压扁后均匀扯开。
- 锅中足量水烧开，下面条煮**2分钟**，捞出过凉水，沥干后放入大盘鸡盘中，拌匀汤汁食用。

---

*小提示：炖煮时水量一次加足，中途尽量不加水，以免影响风味。*
