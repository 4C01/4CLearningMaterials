# 喵喵蛋炒饭

由本喵喵研究出来的原汁原味(没有汁)，粒粒分明，能当菜能当主食的超好吃蛋炒饭www  

## 前期准备

| 材料 | 用量 | 作用与选择 |
| :--- | :--- | :--- |
| 米饭 | 1碗冒尖大米饭 | 隔夜冷冻的yo~ |
| 鸡蛋 | 1个 | 一碗米饭一颗damn~ |
| 鸡精/松茸鲜 | 一大撮 | 好吃的哦~提鲜 |
| 蚝油与植物油 | 蚝油两滴 | 两滴就够~多了口干舌燥 |
| 辅料或盐 | 盐小小一撮，辅料任意 | 火腿午餐肉鱼排蟹柳~但是推荐只撒盐尝尝~ |
| ``可选: 点缀`` | 少量 | 如果强烈强烈不能接受蛋腥味的话... |

## 制作方式

1.  将蛋黄蛋清分离，蛋黄加入米饭中抓碎并充分混合
2.  平锅喷一层薄植物油，将蛋清倒入后用筷子持续搅碎搅散至初步成型
3.  倒入混合好蛋黄的米饭用锅铲铺平，持续翻拌压碎至粒粒分明
4.  米饭松散后，加入极少量的蚝油并翻炒均匀(可以颠锅哦~)
5.  翻炒均匀后，加入辅料或盐(盐捏一点点就够，蚝油也咸，有辅料就不加盐啦)
6.  添加鸡精/松茸鲜快速翻炒均匀，准备出锅
7.  撒上点缀~~(不爱吃葱花的喵喵不撒www)

**炒制时间建议约6分钟内结束(加米饭后)**

## 食用方式

可以将炒饭当成一顿饭~也可以不加盐和辅料夹菜吃~

``还是推荐先尝尝，不夹菜也好吃``
