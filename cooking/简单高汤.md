# 简单高汤

这是使用用天然鲜味食材(香菇、虾皮等)煮成的清水高汤。可给饭菜增加特殊的鲜味  

## 前期准备

| 材料 | 用量 | 作用与选择 |
| :--- | :--- | :--- |
| 干香菇 | 5-8朵 | 普通干香菇即可。 |
| 虾皮 | 1小把（约15克） | 颜色浅干燥无盐的会更加鲜美 |
| 清水 | 约1升（5碗水） | 不加水就不是汤啦~~ |
| ``可选: 配料`` | 1小段胡萝卜或1块玉米 | 可以让汤带一丝甜味 |

## 开始制作

1.  干香菇、虾皮用凉水稍微冲洗一下
2.  将香菇和虾皮放入一个大碗，加入没过它们的凉水浸泡30分钟左右变软  
3.  将泡好的食材连同泡食材的水一起倒入锅中，并加入剩余的清水（胡萝卜/玉米等配料一同加入）
4.  大火煮开后，用勺子撇掉浮沫，转为小火，盖上锅盖留条缝，慢炖**30-40分钟**
5.  煮好后用筛子滤出清澈的汤水，香菇和虾皮可以丢弃（吃了也没事）

## 使用方式

可以用于替代炒菜或煮粥/面的水，提升鲜味

| 使用场景 | 做法与效果 |
| :--- | :--- |
| 煮粥/煮面 | 直接代替水来煮粥或下面条，放一点肉末和青菜，会非常鲜美 |
| 焯蔬菜/烩菜 | 焯西兰花或煮冬瓜时用高汤代替一半水，蔬菜会吸收鲜味 |
| 蒸蛋羹 | 用高汤代替水来调蛋液，蒸出的蛋羹鲜味十足，口感更嫩滑 |
| 做汤底 | 直接作为汤的底料，加入豆腐、蘑菇、青菜等煮熟，只需加一点盐调味即可 |
| 代替料酒腌肉 | 肉类焯水前，可以用少许高汤浸泡一下，有助于去腥增鲜 |

## 储存方式

| 储存方式 | 时间 | 使用提醒 |
| :--- | :--- | :--- |
| **冷藏** | 2-3天 | 尽快用完，保证新鲜 |
| **冷冻（推荐）** | 2-3周 | 将高汤倒入冰格冻成高汤冰块。做菜取几块融化后使用 |
| **真空包装** | 1-2月 | 像储存泡菜/罐头一样使用玻璃罐或真空干净的容器长期不开封储存 |

**不建议过长时间储存，味道会有轻微变化(除非配料太足了www)**
