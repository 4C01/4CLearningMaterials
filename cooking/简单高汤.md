# 🍲 温和万能高汤手册

## 一、这是什么？为什么你需要它？
这是一种用**天然鲜味食材**（香菇、虾皮）煮成的清水高汤。
- **作用**：代替葱、姜、料酒，给饭菜**增加浓浓的鲜味**，让即使只加了盐的菜也很好吃。
- **特点**：**温和、无刺激、无添加**，完全适合你的口味。

## 二、核心配方与制作
### 📝 所需材料
| 材料 | 用量 | 作用与选择 |
| :--- | :--- | :--- |
| **干香菇** | 5-8朵 | **核心鲜味来源**，选普通干香菇即可。 |
| **淡晒虾皮** | 1小把（约15克） | **核心鲜味来源**，选颜色浅、干燥无盐的，味道更纯。 |
| **清水** | 约1升（5碗水） | 基础。 |
| **可选增甜** | 1小段胡萝卜或1块玉米 | 可以让汤带一丝自然甜味，更柔和。 |

### 🔪 制作步骤
1.  **准备食材**：干香菇、虾皮用凉水稍微冲洗一下。
2.  **浸泡食材**：将香菇和虾皮放入一个大碗，加入没过它们的**凉水**，浸泡**30分钟**。让它们变软并初步释放味道。
3.  **开始熬煮**：将**泡好的食材连同泡的水**一起倒入锅中，再加入剩余的清水（或胡萝卜/玉米）。
4.  **火候与时间**：
    - **大火**煮开后，用**勺子撇掉**浮沫。
    - 转为**小火**，盖上锅盖留一条缝，慢炖**30-40分钟**。
5.  **过滤保存**：煮好后，用筛子滤出清澈的汤水，香菇和虾皮可以丢弃（它们的鲜味已经都在汤里了）。

## 三、如何使用这份“液态宝藏”？
你可以像用水一样使用它，但会让食物美味升级！

| 使用场景 | 具体做法与效果 |
| :--- | :--- |
| **煮粥/煮面** | **直接代替水**来煮粥或下面条，比如“香菇高汤粥”，即使只放一点肉末和青菜，也会非常鲜美。 |
| **焯蔬菜/烩菜** | 用水焯西兰花或煮冬瓜时，**用一半高汤代替一半水**，蔬菜会吸收鲜味。 |
| **蒸蛋羹** | 用**放凉的高汤代替水**来调蛋液，蒸出的蛋羹鲜味十足，口感更嫩滑。 |
| **做汤底** | 直接作为汤的底料，加入豆腐、蘑菇、青菜等煮熟，只需加一点盐调味即可。 |
| **代替料酒腌肉** | 肉类焯水前，可以用少许高汤浸泡一下，有助于去腥增鲜。 |

## 四、储存与备忘
| 储存方式 | 时间 | 使用提醒 |
| :--- | :--- | :--- |
| **冷藏** | 2-3天 | 尽快用完，保证新鲜。 |
| **冷冻（推荐）** | 2-3周 | 将高汤倒入**冰格**，冻成“高汤冰块”。每次做菜取几块融化使用，超级方便！ |

**小贴士**：一次可以多做一点，冻成冰块，就是你的“鲜味魔法冰块”了，随时取用。
